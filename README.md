# Pedra, Papel e Tesoura

Projeto desenvolvido em Python que simula o jogo clássico entre o usuário e o computador.

## Como funciona
O usuário escolhe entre pedra, papel ou tesoura, enquanto o computador faz uma escolha aleatória. O sistema compara as opções e exibe o resultado da rodada.

## Tecnologias utilizadas
- Python
- Biblioteca random

## Objetivo
Praticar conceitos de:
- Lógica de programação
- Estruturas condicionais
- Entrada e saída de dados
- Uso de listas e aleatoriedade
